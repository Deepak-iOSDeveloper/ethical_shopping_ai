How EcoMind AI Works

A technical deep-dive into the machine learning models and data science powering your ethical shopping recommendations.

🔬 Technology Stack

🐼
Pandas
Used to load, clean, and manipulate the product dataset. Handles filtering by category, budget, and keyword across 30+ products.
pandas · DataFrame · filtering
🔢
NumPy
Powers array operations for similarity scoring, score normalization, and the cosine similarity computation matrix.
numpy · arrays · linear algebra
🤖
Scikit-learn
Core ML library: RandomForestClassifier for ethical ratings, MinMaxScaler for normalization, cosine_similarity for product similarity.
sklearn · RandomForest · cosine_similarity
🌐
Django
Full-stack web framework serving the AI model through REST API endpoints and beautifully rendered HTML templates.
django · views · templates · REST API

🧮 Composite Score Formula

Every product receives a single composite ethical score, computed as a weighted average of three key dimensions:

# EcoMind Composite Score (0–10)
composite_score = (
  eco_score × 0.40      # 40% — environmental impact
 + ethics_score × 0.40   # 40% — labor & supply chain ethics
 + (10 - carbon_footprint) × 0.20 # 20% — low carbon = higher score
)

⚙️ ML Pipeline

01
Data Ingestion with Pandas
30 real-world ethical products loaded into a Pandas DataFrame. Each product has eco score, ethics score, carbon footprint, price, materials, and sustainability certifications (GOTS, B-Corp, Fair Trade, etc.).
02
Feature Engineering
Composite score computed. Categories label-encoded with Scikit-learn LabelEncoder. Numerical features normalized via MinMaxScaler. Price tiers derived from budget ranges.
03
RandomForest Training
A RandomForestClassifier (100 trees) is trained on [eco_score, ethics_score, carbon_footprint, price] to predict probability that a product is "highly ethical" (score ≥ 8.8). 80/20 train-test split.
04
Content-Based Filtering
Cosine similarity (via NumPy + Scikit-learn) computed between normalized feature vectors to find similar products. This powers the "You Might Also Like" section.
05
Django REST API
The AI model is exposed via Django views and JSON API endpoints. Filters (category, budget, keyword) are applied before ranking by composite score and returning results.
